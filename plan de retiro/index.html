Objetivo: que el futuro valor de Aportación inicial + aportes mensuales (con interés) sea igual a Prestaciones valor futuro. Si el % sale negativo, significa que con la aportación inicial ya cubres el objetivo (tu yo del futuro te manda un sticker 😄).
Ver forma de cálculo (paso a paso)
Sea N = meses hacia adelante, i = tasa mensual del fondo = (tasa anual)/12, y g = incremento anual del sueldo.

0) Objetivo
Target = Prestaciones valor futuro.

1) Separación por antigüedad (servicio pasado vs futuro)
Proporción pasado = Antigüedad actual / Antigüedad esperada.
FV_pasado = Target × proporción pasado.
FV_futuro = Target − FV_pasado.

2) Aportación inicial (hoy) para cubrir el servicio pasado
A0 = FV_pasado / (1+i)^N.

3) Sueldo mensual proyectado
Sueldo(m) = Sueldo₀ × (1+g)^(m/12), para m = 1..N.

4) Aportes mensuales (como % p del sueldo)
FV_aportes = Σ[ p × Sueldo(m) × (1+i)^(N-m) ].

5) Despeje del porcentaje p
p = FV_futuro / Σ[ Sueldo(m)×(1+i)^(N-m) ].

6) Consigna de valores futuros
FV(A0) = A0×(1+i)^N · FV(% aportes) = p×Σ[...] · FV plan = FV(A0)+FV(% aportes)
